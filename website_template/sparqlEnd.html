SPARQL ENDPOINT

REST API to access RVO data
Endpoint:http://adage2.cse.unsw.edu.au:8005/v1/graphs/sparql

Path Parameter: query=“<URLencodedSPARQLquery>”

Authentication (basic):

  • Username: guest
  • Password: guestUNSW

Sample CURL request:

curl --basic --user guest:guestUNSW -i -X GET -H "Accept: application/sparql- results+json" http://adage2.cse.unsw.edu.au:8005/v1/graphs/sparql?query="<URL encoded query>"
